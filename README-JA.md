`improvie` は [Tauri](https://tauri.app/) で構築された軽量デスクトップアプリケーションで、ローカルビデオやオーディオファイルを簡単に管理できます。メディアの閲覧、カスタムプレイリストの作成、スムーズな再生を、クリーンで直感的なインターフェイスで実現します。

## ✨ 機能

- 📂 **ローカルメディア管理**
  マシンに保存されたビデオ/オーディオファイルをインポートして整理。

- 🎵 **プレイリスト作成**
  ローカルメディアからプレイリストを素早く作成・編集。

- 🎧 **軽量メディアプレーヤー**
  ローカルコンテンツに最適化されたシームレスな再生体験。

- ⚡ **Tauriで構築**
  ネイティブ感覚の高速で安全なアプリケーション。

- 🌐 **多言語対応**
  日本語と英語をサポートし、ユーザーは好みの言語で使用可能。

- 🎨 **テーマカスタマイズ**
  アプリの見た目をカスタマイズ可能。

## 📥 インストール

### 推奨環境

- **OS**: Windows 10/11, macOS 11+, Ubuntu 20.04+
- **RAM**: 2GB以上推奨
- **ディスク容量**: 100MB以上の空き容量

### ダウンロード

最新リリースは[Githubリリースページ](https://github.com/improvie/improvie/releases/latest)からダウンロードできます。
